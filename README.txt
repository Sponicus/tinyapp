# TinyApp Project

TinyApp is a web application built with Node and Express that allows users to shorten long URLs.

## Final Product

!["screenshot of URLS page"](https://github.com/Sponicus/tinyapp/blob/master/docs/TinyAppURLs.png)

!["screenshot of registration page"](https://github.com/Sponicus/tinyapp/blob/master/docs/TinyAppRegistration.png)

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
- Use the full URLs when creating new shortened URLs.
